# Library Anime

## Why is Library Anime?

Library Anime is an application developed with Flutter to search for anime to watch by name, genres, available streaming platforms, and ratings. It also gives the user the ability to rate the anime, write a review, save to watch later, track anime to be notified when a new episode comes out, and profile customization.


## What problem does it seek to solve?

Anime content is distributed across more 8 platforms, which causes users to look for a series to watch for approximately 20 to 30 minutes per session, the visual content separate in external web sites

## What solutions does Library Anime offer?

Library Anime offers a unified catalog, an advanced filter by genres, age, name, rating, availability, it also offers a history and personalized recommendations for users, along with reviews, videos, and other visual content of the series, and finally news from the world of anime in one place
